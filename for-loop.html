for文：条件を詳細に指定する「構文」

for文は、JavaScriptの「文法」として用意されている、最も基本的な繰り返しの命令です。
「開始値、終了条件、増減値」を個別に記述します。

1. 構文の仕組み

for文は、3つの要素を指定してループを制御します。

for (初期化; 条件式; 更新式) {
    // 繰り返したい処理
}

3つの構成要素

① 初期化 (Initialization)
ループ開始時に1回だけ実行されます。カウンター変数（例: let i = 0;）を定義します。
② 条件式 (Condition)
各ループの開始前に評価されます。この条件が true の場合のみ、ブロック内の処理が実行されます。
例: i < 3;
③ 更新式 (Update)
各ループの終了後に実行されます。カウンター変数を更新します。
例: i++

覚えておきたい言葉

カウンター変数 (Counter Variable)
ループの回数を数えるための変数のこと。慣習として i (indexの頭文字) がよく使われます。
ブロック (Block)
{ ... } で囲まれた範囲のこと。「この中の処理を繰り返すよ」というグループ分けです。
インクリメント (Increment)
変数の値を 1 増やすこと。i++ は i = i + 1 と同じ意味です。
const animals = ['Dog', 'Cat', 'Bird'];

for (let i = 0; i < animals.length; i++) {
    console.log(animals[i]); // インデックスを使って要素を取り出す
}

2. for文の機能（メリット）

for文は、ループの挙動を細かく制御できるため、以下の機能が利用できます。
これらは forEach では利用できない機能です。

🛑 中断 (break)

条件を満たした時点で、ループを終了できます。

⏭️ スキップ (continue)

特定の条件の時だけ処理を行わず、次のループへ進むことができます。

⏳ 待機 (await)

async/await を使用して、非同期処理の完了を待機してから次のループへ進むことができます。

🔄 柔軟な制御

逆順でのループ (i--) や、2つ飛ばし (i+=2) など、カウンター変数を自由に操作できます。
