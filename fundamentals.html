基礎知識

プログラミングにおいて、「同じ処理を繰り返す」ことは非常に重要です。
例えば、リストにある全ての項目を表示したり、全ての数値に1を足したりする場合です。

1. 配列 (Array) とは

複数のデータをまとめて管理するための箱のようなものです。
[] で囲み、カンマ , で区切ってデータを入れます。

例：動物のリスト

const animals = ["Dog", "Cat", "Rabbit"];

このリストには、3つのデータ（要素）が入っています。

2. ループ (Loop) とは

「配列の中身を順番に取り出して、何かをする」処理のことです。
例えば、「動物リストの全員に挨拶する」という処理を考えてみましょう。

やりたいこと：

  • 1回目 (インデックス0)：Dog を表示する
  • 2回目 (インデックス1)：Cat を表示する
  • 3回目 (インデックス2)：Rabbit を表示する
🐶 Dog 🐱 Cat 🐰 Rabbit

同じコードを複数回記述するのではなく、1つの命令でまとめて実行するためにループを使用します。

3. 構文とメソッドの違い

for 文と forEach は、どちらもループ処理を行いますが、仕組みが異なります。

構文 (Syntax)

言語のルールそのものです。
if や for などがこれにあたります。
処理の細かい手順を自分で記述する必要があります。

メソッド (Method)

あらかじめ用意された機能です。
forEach は配列が持っている機能です。
「何をしたいか」を指定するだけで実行されます。

4. 重要な用語

次のページに進む前に、用語を整理します。

変数 (Variable)
データを格納するための領域です。
const (定数) や let (変数) を使って宣言します。
要素 (Element)
配列に入っている1つ1つのデータのことです。
例："Dog" や "Cat"
インデックス (Index)
配列の中での順番を示す番号です。
0から始まります (0, 1, 2...)。
